Detailed Description

This class represents a binary code word.

The data structure consists of a vector from STL containing either 32-bit or 64-bit words, depending on the target architecture. Data can be added and accessed in different ways. A code word can be considered as a bit, 32-bit word or 64-bit word (little endian). The length of a code word refers to the bit representation.

Examples:

allinone.cpp, search.cpp, sha1me.cpp, and shortening.cpp.
