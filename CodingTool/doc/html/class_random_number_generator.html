Detailed Description

This class provides access to a random number generator.

The CodingTool does not implement its own random number generator. Instead it is using the generator from Makoto Matsumoto and Takuji Nishimura which was ported to C++ by Jasper Bedaux (see http://www.bedaux.net/mtrand/).


The purpose of this interface is to make the type of random number generator easy exchangeable.
